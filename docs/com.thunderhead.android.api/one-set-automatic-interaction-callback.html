oneSetAutomaticInteractionCallback

fun oneSetAutomaticInteractionCallback(oneInteractionPath: OneInteractionPath, init: OneAutomaticInteractionCallback.() -> Unit): Unit

Set a callback in order to be notified of automatic interaction requests.

To remove call oneRemoveAutomaticInteractionCallback.

Example:

oneSetAutomaticInteractionCallback(OneInteractionPath(URI("https://server.com"))) {
 onSuccess { response ->
     // perform custom action
     response.process()
 }

 onError { sdkError ->
     Log.d(TAG, "SdkError: ${sdkError.errorMessage}")
 }

 onFailure { apiError ->
     Log.d(TAG, "ApiError: ${apiError.errorMessage}")
 }
}
